# 🔐 Cryptography Lab Experiments (Python)

This repository contains Python implementations of classical and modern cryptographic algorithms developed as part of the **Cryptography and Network Security (CNS) Laboratory**.

The programs are written in **Python 3** and follow the experiments prescribed in the laboratory manual.

# 💻 Technologies Used

- Python 3.x

- VS Code

---

# 📦 Requirements

Install Python 3.

For the DES experiment, install the required package:

```bash
pip install pycryptodome
```

---

# ▶️ Running the Programs

Open the **Lab Experiments** folder.

Run any experiment using:

```bash
python exp1.py
```

Similarly,

```bash
python exp2.py
python exp3.py
python exp4.py
python exp5.py
python exp6.py
python exp7.py
```

# 🎯 Learning Outcomes

After completing these experiments, students will be able to:

- Understand classical encryption techniques.
- Implement substitution and transposition ciphers.
- Perform encryption and decryption using symmetric algorithms.
- Understand public-key cryptography concepts.
- Gain practical experience in implementing cryptographic algorithms using Python.
